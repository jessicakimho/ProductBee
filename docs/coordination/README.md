# Coordination Directory

This directory is used for coordination between Backend and Frontend agents.

## Purpose

When Backend Agent needs Frontend Agent's input or when Frontend Agent needs Backend Agent's input, create a request file here.

## File Naming Convention

- `backend-request-[feature-name].md` - Requests from Backend Agent
- `frontend-request-[feature-name].md` - Requests from Frontend Agent

## Request Format

Each request should include:
- **Feature/Issue**: What needs coordination
- **Context**: Why this coordination is needed
- **Requested Action**: What the other agent should do
- **Blocking**: Whether this blocks progress
- **Timeline**: When this is needed
